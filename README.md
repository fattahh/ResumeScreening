## 📄 What is ResumeScreening?

ResumeScreening is a tool that helps you quickly review resumes. It uses smart technology to read and sort resumes in seconds. It can replace hours of manual work with automatic analysis. This lets you find the best candidates faster and save time.

The system was built with powerful tools like LangChain, LangGraph, and FastAPI. It works with a big language model that understands text deeply. Instead of spending 3-5 minutes on each resume, you spend 3-5 seconds.

## 🎯 How to Use ResumeScreening

This section explains the main steps for using the software.

### 1. Add Resumes

Click the "Add Resumes" button. You can select one or multiple resume files from your computer. Supported file types include PDF, DOCX, and TXT.

### 2. Smart Parsing

Once added, the system reads each resume to find important information like name, education, skills, and work history. This parsing happens automatically and quickly.

### 3. Filter Candidates

You can set criteria for filtering resumes. For example, you might want candidates with experience in a certain field or specific skills. Enter the criteria, then click "Filter". The program will show a list of matching resumes.

### 4. Ask Questions

Use the built-in Q&A feature to ask about resumes. For example, you can ask "Who has experience with project management?" or "Which candidates graduated after 2018?" The system will answer based on the parsed data.

### 5. Export Results

After filtering or reviewing, export the list to a CSV or Excel file for your records or sharing.
